## 🔍 **PARA ID DO ITEM (admins apenas):**

### **1. ItemInstance.java**
**Caminho provável:**
```
java/ext/mods/gameserver/model/item/instance/ItemInstance.java
```

**O que procurar:**
```java
public String getName() {
    return _item.getName();
}

// OU

public String getItemName() {
    return _name;
}

// Mude para:
public String getName(Player player) {
    if (player != null && ItemHelper.canSeeItemId(player)) {
        return _item.getName() + " (ID: " + getItemId() + ")";
    }
    return _item.getName();
}
```

## 🎯 **PASSO A PASSO PRÁTICO:**

### **Para começar RÁPIDO:**
1. **Vá para:** `EnterWorld.java` (já sabemos onde está)
2. **Encontre:** `player.onPlayerEnter();` (linha ~362)
3. **Adicione DEPOIS:** 
   ```java
   // Dá itens iniciais ao jogador
   StartingItemsManager.getInstance().giveStartingItems(player);
   ```
4. **Compile:** `ant compile`
5. **Teste:** Crie novo personagem

### **Para ID do item:**
1. **Procure por:** `ItemInstance.java`
2. **Encontre método:** `getName()` 
3. **Adicione parâmetro Player:** `getName(Player player)`
4. **Use:** `ItemHelper.getFormattedItemName(player, this)`

## 💡 **DICA PARA ENCONTRAR:**

### **Use grep para procurar:**
```bash
# Para encontrar ItemInstance
find . -name "ItemInstance.java" -type f

# Para encontrar onde mostra nome do item

find . -name "EnterWorld.java" -type f
```
