## 📋 **PASSO 1: CONFIGURAÇÃO JÁ FEITA**

✅ **Configuração adicionada** em `players.properties`:
- `ShowItemIdForAdminOnly = True` - Só admins veem ID
- `MinimumAccessLevelToSeeItemId = 1` - Nível mínimo admin
- `StartingItems_*` - Itens iniciais configurados

## 📁 **PASSO 2: ARQUIVOS CRIADOS**

✅ **1. `ItemHelper.java`** - Classe para formatar nome do item
✅ **2. `StartingItemsManager.java`** - Classe para dar itens iniciais

## 💡 **PASSO 5: PERSONALIZAR**

### **Personalizar IDs visíveis:**
1. Abra `players.properties`
2. Mude `ShowItemIdForAdminOnly = False` para todos verem
3. Mude `MinimumAccessLevelToSeeItemId = 3` para admins nível 3+

### **Personalizar itens iniciais:**
1. Abra `players.properties`
2. Edite as linhas `StartingItems_*`
3. **Formato:** `itemId:quantidade,itemId:quantidade,...`
4. **Exemplo:** `57:5000,1835:50,17:1`

## 🚀 **PASSO 6: IMPLEMENTAÇÃO RÁPIDA**

**Para testar rápido, faça isto:**

1. **Copie os arquivos** para a pasta `java/ext/mods/gameserver/`
2. **Encontre o método** `onEnterWorld()` em `EnterWorld.java`
3. **Adicione esta linha** no final do método:
   ```java
   StartingItemsManager.getInstance().giveStartingItems(player);
   ```
4. **Compile:** `ant compile`
5. **Teste:** Crie novo personagem e veja os itens
